# Sri Balasubramaniyam Timber Mart

## Project Description

This project is a **MERN Stack–based web application** developed as a **consultancy project** for **Sri Balasubramaniyam Timber Mart**, a furniture and timber shop.
The application provides a digital platform to showcase products and manage shop information efficiently.

## Technology Stack

**Frontend:** React.js, HTML, CSS, JavaScript
**Backend:** Node.js, Express.js
**Database:** MongoDB
**Tools:** VS Code, Git, Postman

## Features

* Responsive user interface
* Furniture and timber product display
* RESTful APIs for backend operations
* Secure data storage using MongoDB

## Setup Instructions

1. Clone the repository
2. Install dependencies in both `client` and `server` folders
3. Configure MongoDB in `.env`
4. Run backend and frontend servers
